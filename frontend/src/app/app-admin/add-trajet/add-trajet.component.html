Ajouter un Trajet

Remplissez les détails du trajet pour l'ajouter à la base de données.

le nom de trajet est requis. le nom de trajet doit contenir au moins 3 caractères. le nom de trajet ne peut pas dépasser 30 caractères.
Le départ est requis. Le départ doit contenir au moins 3 caractères. Le départ ne peut pas dépasser 30 caractères.
L'arrivée est requise. L'arrivée doit contenir au moins 3 caractères. L'arrivée ne peut pas dépasser 30 caractères.
La durée est requise. La durée doit être d'au moins 1 heure.
La distance est requise. La distance doit être d'au moins 1 km.
